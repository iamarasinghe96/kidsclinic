
# RECEPTIONIST LAPTOP SETUP

## Installation Steps:

1. **Copy this entire RECEPTIONIST folder to D:\RECEPTIONIST**

2. **Right-click on `install.bat` and select "Run as administrator"**

3. **Double-click `START_CLINIC.bat` to start the system**

## Daily Usage:
- Double-click **START_CLINIC.bat** to start the clinic system
- Browser will open automatically to the receptionist interface
- System runs on your network at IP: 192.168.1.11

## Troubleshooting:
- If browser doesn't open: Go to http://localhost:5000
- If server won't start: Check if port 5000 is free
- If Python missing: Install from python.org
