
# CONSULTANT LAPTOP SETUP

## Installation Steps:

1. **Copy this entire CONSULTANT folder to D:\CONSULTANT**

2. **Right-click on `install.bat` and select "Run as administrator"**

3. **Double-click `START_CONSULTANT.bat` to access the system**

## Daily Usage:
- Double-click **START_CONSULTANT.bat** to open consultant view
- Browser opens automatically to the consultant interface
- Connects to receptionist system at: 192.168.1.11

## Note:
- This laptop only views patient information
- All patient management is done on receptionist laptop
- No server runs on this laptop - it just opens a browser

## Troubleshooting:
- If can't connect: Check if receptionist system is running
- Check network connection between laptops
- Verify IP address: 192.168.1.11
