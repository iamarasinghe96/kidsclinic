# Cleaned Project Structure

The project has been simplified to these essential files:

## Core Application Files
- `app.py` - Flask application setup
- `main.py` - Application entry point  
- `models.py` - Database models
- `routes.py` - Route handlers
- `pyproject.toml` - Dependencies

## Configuration
- `.replit` - Replit configuration
- `replit.md` - Project documentation

## Database
- `instance/clinic.db` - SQLite database file

## Frontend
- `templates/base.html` - Base template
- `templates/receptionist.html` - Receptionist interface
- `templates/consultant_simple.html` - Consultant interface  
- `templates/report.html` - Reports interface
- `static/script.js` - JavaScript
- `static/style.css` - Custom styles

## Removed Files
All unnecessary files have been removed:
- Old backup folders (CONSULTANT_LAPTOP, RECEPTIONIST_LAPTOP)
- Export and migration scripts
- Duplicate templates
- Cache and temporary files
- Archive files

The project is now ready for a clean rebuild with the simplified structure.
